Mise en ligne avec GitHub Pages :
1. Créer un dépôt GitHub.
2. Ajouter index.html, style.css et vos PDF.
3. Settings > Pages > Deploy from a branch > main / root.
4. Créer un QR code vers l'URL GitHub Pages.

À modifier :
- Remplacer le bloc Troisième poster.
- Remplacer le lien ResearchGate par votre profil personnel.
- Si vous voulez des PDF directs, remplacez les liens ResearchGate par poster-2019.pdf, poster-2018.pdf, poster-3.pdf.
